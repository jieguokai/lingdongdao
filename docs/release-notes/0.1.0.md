# Codex Lobster Island 0.1.0

## Highlights

- Initial public release of the macOS floating status island and menu bar app
- Mock-driven Codex state visualization with `idle`, `running`, `success`, and `error`
- Provider architecture prepared for future real Codex integrations

## Included in this version

- top floating island window
- menu bar control surface
- pixel lobster placeholder mascot and state animation
- sound, mute, settings, and launch-at-login support
- mock, process, log, and socket provider paths
- release packaging, Sparkle integration, and GitHub release automation templates

## Known limitations

- real Codex integration is still provider-based rather than a first-party Codex protocol
- notarized public binaries still depend on your own Apple signing credentials
