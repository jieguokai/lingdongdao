1. # Product Spec

   ## Product
   Codex Lobster Island

   ## Summary
   A native macOS top floating status island + menu bar utility that visualizes Codex work status.

   ## Core statuses
   - idle
   - running
   - success
   - error

   ## MVP requirements
   - menu bar extra
   - floating top island
   - mock-driven status transitions
   - lobster mascot placeholder
   - per-state animation
   - sound notifications for success/error
   - settings for mute/show island/start at login

   ## Non-goals for MVP
   - no real Codex binding yet
   - no cloud sync
   - no account system
   - no multi-agent management
   - no analytics backend
