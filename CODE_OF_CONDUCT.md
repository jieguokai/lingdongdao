# Code of Conduct

## Our standard

We want Codex Lobster Island to be a practical, respectful, and welcoming open source project.

Examples of behavior that support a healthy community:

- giving constructive technical feedback
- assuming good intent before escalating
- focusing on reproducible problems and concrete fixes
- respecting different experience levels and backgrounds

Examples of unacceptable behavior:

- harassment, insults, or personal attacks
- doxxing or exposing private information
- hostile or inflammatory trolling
- repeated bad-faith disruption of issues or pull requests

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, and pull requests that violate this policy.

## Reporting

If someone is violating this code of conduct, contact the repository maintainers privately rather than escalating in public issue threads.
